# Migrations Documentation

Thư mục này chứa các tài liệu về database migrations.

## Danh sách các file

- `RUN_MIGRATION_AUDIT_COLUMNS.md` - Chạy migration audit columns
- `RUN_MIGRATION_TASK_COMMENTS_REPLY.md` - Chạy migration task comments reply

## Thư mục liên quan

- `database/migrations/` - Chứa các file SQL migration
- `supabase/migrations/` - Chứa các file Supabase migration
